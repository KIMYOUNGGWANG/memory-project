# Memory project - front

## 목적

- 사람들이 추억을 저장할수 있는 앨범을 만들어볼수 있도록
- 사람들과 공유를 쉽게 하도록
- 시간 장소 순으로 위치까지 볼수 있도록

## 기술 스택

- Reactjs
- typescript
- react query
- rxjs... 가능할지는 모르겠음..

## 개인적 목적

- 좀더 단단한 프로젝트 보일러플레이트를 만들어보고자 이것저것 적용해가며 테스트
- webpack babel 등을 이용해서 alias 적용
- 추가적으로 좀더 편한 좀더 단단한 프로젝트는 무엇일까 고민해보는 프로젝트
